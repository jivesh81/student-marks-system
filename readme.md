# Student Marks Management System

A simple Python program to manage student marks.

## What does it do?

This program helps you store and manage student marks. You can add students, view their marks, find the topper, update marks, delete students, calculate class average, and search for specific students.

## How to run?

1. Make sure you have Python installed on your computer
2. Download the code file
3. Open terminal or command prompt
4. Run the command: python filename.py
5. Follow the menu options

## Features

- Add student marks
- View all students and their marks
- Find the topper student
- Update existing marks
- Delete a student record
- Calculate class average
- Search for a specific student
- Easy to use menu system

## Requirements

- Python 3.x
- No additional libraries needed

## How to use?

When you run the program, you will see a menu with 8 options. Just enter the number of your choice and follow the instructions.

The program will keep running until you choose option 8 to exit.

## Notes

- Student names are not case sensitive
- Marks must be between 0 and 100
- All data is stored in memory and will be lost when you exit the program
